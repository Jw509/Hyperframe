# Sports Card Opening — Title Workflow

Hyperframes-driven pipeline. Source video + comp screenshots + prices in → transparent ProRes title MOVs out. Direction for cuts and final video happens in chat with Claude, who builds the final composition in Hyperframes.

## One-time setup (done)

- Node.js 22+
- FFmpeg (Gyan build, installed via winget)
- Hyperframes project at `EditHyper/cards/`

## Per-video loop

1. **Drop the source video** into `EditHyper/` (project root) and reference it from a brief in `briefs/<slug>.json`.

2. **Capture eBay comp screenshots** — Win+Shift+S for each card you'll call out.
   Tight crop: item title, the green "Sold" price, sold date. Save as PNG.
   Drop them in `cards/comps/<slug>/`. Filenames can be anything — reference them in the brief via `comp_image`.

3. **Edit the brief** — `briefs/<slug>.json`:

   ```json
   {
     "source": "<slug>.mp4",
     "output_fps": 60,
     "reveals": [
       { "at": "0:17", "value": "$3.25", "comp_image": "maye.png" },
       { "at": "0:29", "value": "$25",   "comp_image": "pq.png",   "money_shot": true },
       { "at": "0:47", "value": "$1",    "comp_image": "mev.png" }
     ]
   }
   ```

   Fields per reveal:
   - `at` *(required)* — timestamp in the source video (`MM:SS` or seconds)
   - `value` *(required)* — the big number, e.g. `"$420"`
   - `comp_image` *(optional)* — filename inside `cards/comps/<slug>/`
   - `money_shot` *(optional)* — flags the highest-comp card; gets longer hold time

4. **Render titles** (from project root):

   ```powershell
   npm run build:edit briefs/<slug>.json
   ```

   Output: `cards/renders/<slug>/landscape/NN-slug.mov` — ProRes 4444 with full alpha, comp screenshot baked in.

5. **Tell Claude what you want next** — cuts, pacing, music, anything specific about the final video. Claude builds a Hyperframes composition that ingests the source, applies the cuts, lays the title MOVs on top, and renders the final MP4.

## Files

- `cards/index.html` — landscape 1920x1080 title composition
- `cards/compositions/portrait.html` — vertical 1080x1920 variant
- `cards/cards.json` — generated by `build:edit` from the brief; you don't edit it directly
- `cards/cards.schema.json` — JSON Schema for cards.json
- `cards/scripts/render-cards.mjs` — batch title renderer
- `cards/comps/<slug>/` — eBay sold screenshots
- `briefs/<slug>.json` — per-video input
- `scripts/build-edit.mjs` — brief → cards.json → render titles
